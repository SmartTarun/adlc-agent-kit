Kavya's Gemini Prompt Template

Kavya constructs the prompt from the product brief fields:

const prompt = [
  `Style: ${productBrief.userPersonas} -- ${brandStyle}`,
  `Use case: ${imageUseCase}`,
  `Palette: dark, #0d1117 bg, accent ${accentColor}`,
  `Format: ${width}x${height}px, ${format}`,
  `No text overlays. No watermarks.`
].join('. ');

Gemini API Call (Node.js)

Kavya's script calls Gemini Imagen and saves the output:

const { GoogleGenerativeAI } = require('@google/generative-ai');
const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);
const model = genAI.getGenerativeModel({ model: 'gemini-2.0-flash-exp' });
const result = await model.generateContent([prompt]);
// extract base64 image → upload to S3 → return CDN URL

Vikram's S3 bucket (IaC)

Terraform creates the assets bucket. Kavya writes to it. Rohan reads CDN URLs from it.

Keerthi's QA Checks

After Rohan integrates, Keerthi validates:

-- Image renders at correct dimensions
-- Alt text / aria-label present on <img>
-- Loads in < 2s on slow-3G (Lighthouse)
-- Dark mode: no white background bleed
-- No NSFW / brand-policy violations
Key files:   kavya-gemini.js  |  .env: GEMINI_API_KEY + AWS_ACCESS_KEY_ID  |  /infra/modules/ecs/main.tf (S3 assets bucket)  |  /frontend/src/tokens/tokens.css (CSS vars Kavya writes)
